## Requirements
* Java 1.6
* RXTX (Ubuntu: apt-get install librxtx-java)
* OpenGL 2.0

## Usage
Start the Program: `java -jar grblgui.jar <your g-code file or directory> <arduino COM-port>`

Example: `java -jar grblgui.jar /home/cody/gcode/ /dev/ttyACM0`
